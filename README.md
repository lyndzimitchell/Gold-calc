# Gold Calc Version 2

Version 2 adds:
- Calculator tab
- Inventory tab
- Scrap / melt inventory bucket
- Needs research inventory bucket
- Optional item photo
- Weight, karat, notes, price paid
- Current melt value per item
- Total melt value by bucket
- Local device/browser storage
- JSON backup export

Upload these files to GitHub Pages:
- index.html
- manifest.json
- service-worker.js
- icon.svg
- README.md

Important: inventory is stored locally on the device/browser. Use Export Backup JSON before clearing browser data or changing phones.
