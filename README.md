# Gold Buyer Calculator

A tiny phone-friendly gold buying calculator with:
- live XAU/USD gold spot lookup
- manual spot override
- melt value
- maximum offer
- expected refiner payout
- scrap profit / ROI
- resale-as-jewelry comparison
- installable home-screen app behavior

## Free hosting option: GitHub Pages

1. Create a free GitHub account.
2. Make a new public repository named `gold-calc`.
3. Upload these files into the repository:
   - index.html
   - manifest.json
   - service-worker.js
   - icon.svg
4. Go to Settings → Pages.
5. Under "Build and deployment," choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
6. Save.
7. Wait a minute or two for GitHub to publish the site.
8. Open the published link in Safari on your iPhone.
9. Tap Share → Add to Home Screen.

## Important

Live spot is a guide only. Always verify current market price and refiner terms before paying.
